# 💪 Gestão de Treinos - PWA

Uma app para registar e acompanhar treinos semanais com cobertura muscular completa!

## 📱 Como Usar como App

### No Telemóvel (Chrome, Edge, Firefox)

1. Abre o link da app no telemóvel
2. Clica no ícone **⋮** (três pontos) no canto superior direito
3. Seleciona **"Instalar app"** ou **"Adicionar à tela inicial"**
4. Pronto! A app funciona como nativa! 🚀

### No iPhone (Safari)

1. Abre em Safari
2. Clica no ícone de **Partilha** (⬆️)
3. Seleciona **"Adicionar à Tela Inicial"**
4. Pronto! Funciona como app nativa

## ✨ Funcionalidades

- ✅ **Registo de Treinos** - Conta as sessões de cada grupo muscular
- ✅ **Cobertura Semanal** - Vê o progresso visual da tua semana
- ✅ **Grupos Musculares** - 12 grupos diferentes com exercícios sugeridos
- ✅ **Histórico** - Acompanha a evolução das tuas semanas
- ✅ **Offline** - Funciona sem internet! 📡❌
- ✅ **Dados Locais** - Tudo guardado no teu telemóvel (privacidade garantida) 🔒

## 🎯 Como Funciona

1. **Vê a semana atual** (formato ISO: YYYY-WXX)
2. **Clica nos botões +/−** para adicionar/remover sessões de cada grupo
3. **A barra de progresso** mostra quantos grupos já treinaste
4. **"Por Treinar"** lista os grupos que ainda faltam
5. **"Histórico"** guarda todas as semanas anteriores

## 📂 Ficheiros

- `index.html` - App principal
- `manifest.json` - Configuração PWA
- `service-worker.js` - Suporte offline

## 📋 Grupos Musculares

1. **Peito** - Máquina de empurrar para a frente
2. **Costas** - Puxador, Remada
3. **Ombros** - Máquina de ombros
4. **Bíceps** - Halteres
5. **Tríceps** - Corda ou halteres
6. **Quadríceps** - Prensa, Agachamento
7. **Isquiotibiais** - Flexora
8. **Glúteos** - Afundos, Hip Thrust
9. **Abdutores** - Abdutora
10. **Core - Abdominais** - Crunches, Prancha
11. **Core - Lombar** - Extensões lombares
12. **Cardio** - Corrida, Fiit, Jump

## 💡 Dicas

- A app guarda tudo localmente no teu telemóvel
- Funciona 100% offline após primeira visita
- A semana reseta automaticamente (formato ISO)
- "Próxima Semana" avança 7 dias
- Os dados nunca saem do teu telemóvel
- Partilha o link com amigos para eles também usarem!
